======================
SCVersioning |version|
======================

A Sphinx extension that lets you build Sphinx docs for all versions of your project without needing special hosting
services.

+--------------------+--------------------+
| **A Few Examples**                      |
+--------------------+--------------------+
| |alabaster|        | |sphinx_rtd_theme| |
|                    |                    |
| *alabaster*        | *sphinx_rtd_theme* |
+--------------------+--------------------+
| |classic|          | |nature|           |
|                    |                    |
| *classic*          | *nature*           |
+--------------------+--------------------+
